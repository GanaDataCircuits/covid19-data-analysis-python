# covid19-data-analysis-python
COVID-19 data analysis, visualization, and time series forecasting using Python, Plotly, and Prophet.
# 🌍 COVID-19 Trend Analysis and Forecasting

## 📌 Overview
This project analyzes global COVID-19 data to understand infection trends, recovery patterns, and predict future cases using time series forecasting.

## 🚀 Features
- Data cleaning and preprocessing using Pandas
- Interactive visualizations using Plotly
- Global COVID-19 choropleth map
- Trend analysis of confirmed, recovered, and active cases
- Time series forecasting using Facebook Prophet

## 📊 Technologies Used
- Python
- Pandas
- NumPy
- Plotly
- Prophet

## 📈 Key Insights
- Identified countries with highest active cases
- Visualized global spread using maps
- Predicted future COVID-19 cases for the next 7 days
